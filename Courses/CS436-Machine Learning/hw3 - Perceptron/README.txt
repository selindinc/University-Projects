!!!!!!!!!!!!!!!!!

To run the dataset without removing stop words please set the stop_flag to 1.To run the dataset by removing the stop words please set the stop_flag to 0. stop_flag is at the beginning of Perceptron.py file.

To adjust the number of iterations and learning rate, please change the global values at the beginning of the main.py file.

Also, please adjust the directories according to your own file at the beginning of the Perceptron.py and main.py files.


HOW TO COMPILE

python3 main.py
