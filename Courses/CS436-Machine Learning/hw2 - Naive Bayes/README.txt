To run the dataset without removing stop words please set the stop_flag to 1.To run the dataset by removing the stop words please set the stop_flag to 0. stop_flag is at the beginning of NaiveBayes.py file.

Also, please adjust the directories according to your own file at the beginning of the NaiveBayes.py and main.py files.


HOW TO COMPILE

python3 main.py

Please make sure that nltk is installed on your computer.
